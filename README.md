# Document Object Model SBA
This project is created with Javascript for the purpose of showing an understanding of DOM manipulation, browser events, and web application fundamentals. This application allows users to keep track of cats up for adoption by adding new cats, or marking them as adopted.

# User Stories
* As a user, I want to add pets to the tracker so that I can keep record of animals available.
* As a user, I want to view pet infomation so that I can see details about pets.
* As a user, I want to adopt a cat so that I can mark it as no longer available
* As a user, I want the application to confirm my adoption choice so that I do not accidently adopt a cat

## Users can
* View a list of available cats
* Add a new cat using the form
* Enter a cat's name, age, and breed
* Mark a cat as adopted
* See adopted cats displayed differently

## Features
* Uses DOM manipulation with createElement() and appendChild()
* Uses event listeners for form submiion and button clicks.
* Uses arrays, objects, loops, and functions for pet data.
* Uses DocumentFragment to display cay information
